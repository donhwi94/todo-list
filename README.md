# Overview
댓글 기능이 포함된 Todo-list API입니다

# Features
- 할일은 작성 및 디테일 보기와 수정/삭제가 가능하다
- 할일은 최신 등록 날짜로 정렬한다
- 할일은 완료 여부를 통해 필터가 가능하다
<br/>
- 할일에 댓글을 등록할 수 있다
- 댓글은 할일 디테일 보기 페이지에서 볼 수 있다
- 댓글은 수정/삭제가 가능하다
- 댓글은 최신 등록 날짜로 정렬한다 

# Tech Stack

- framework : Django
